
# Password Generator 🔐

A simple Python project that generates secure random passwords based on user preferences.

This program allows users to choose:
- Number of letters
- Number of symbols
- Number of numbers

Then it creates a randomized strong password instantly.

---

## 📌 Features

- User chooses password length composition
- Random letters, symbols, and numbers generated
- Password characters shuffled for better security
- Beginner-friendly Python project
- Demonstrates loops, lists, random module, and string handling

---

## 🛠 Technologies Used

- Python 3
- random module


# Caesar Cipher (Python)

A simple and interactive command-line Caesar Cipher program built using Python.

This project allows the user to encrypt and decrypt messages by shifting letters in the alphabet by a given number. It supports both encoding and decoding operations and allows repeated usage in a loop. Non-alphabet characters such as numbers and symbols remain unchanged.

The project demonstrates Python fundamentals such as loops, conditionals, string manipulation, user input handling, and modular programming using multiple files.

## Features

- Encrypt messages using Caesar Cipher logic  
- Decrypt messages back to original text  
- Supports user input for message and shift value  
- Handles wrap-around using modulo operation  
- Preserves numbers and special characters  
- Allows multiple runs using a loop  
- Displays ASCII art at the start  

## Technologies Used

- Python  
- String manipulation  
- Loops  
- Conditional statements  
- User input handling  
- Modular Python files

## How It Works

1. The program displays a logo using ASCII art.
2. The user chooses whether to encode or decode a message.
3. The user enters the message.
4. The user provides a shift number.
5. Each letter in the message is shifted based on the given number.
6. The program prints the encrypted or decrypted result.
7. The user can choose to restart the program.

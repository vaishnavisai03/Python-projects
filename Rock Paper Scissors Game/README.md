# ✊✋✌️ Rock Paper Scissors Game (Python)

A simple command-line Rock Paper Scissors game built using Python.  
This project allows the user to play against the computer with ASCII visuals.

---

## 🚀 Features

- User vs Computer gameplay  
- Random computer choice  
- ASCII art for rock, paper, and scissors  
- Input validation  
- Simple and interactive CLI interface  

---

## 📂 Project Structure

Rock Paper Scissors Game/  
│  
├── Game.py      # Main game logic  

---

## 🧠 How the Game Works

- User selects:
  - 0 → Rock  
  - 1 → Paper  
  - 2 → Scissors  
- Computer randomly selects a choice  
- Game compares both selections:
  - Same → Draw  
  - Winning combinations → You win  
  - Otherwise → You lose  

---

## ▶️ How to Run

1. Clone the repository:
   git clone https://github.com/your-username/python-projects.git  

2. Navigate to the project folder:
   cd "Rock Paper Scissors Game"  

3. Run the game:
   python Game.py

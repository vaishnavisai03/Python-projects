# Rock-Paper-Scissors
A game well known by everyone and almost each one of them played this game at least once in their lifetime...

This project is a command-line Rock Paper Scissors game built using Python. The program allows a user to play the classic game against the computer. The user selects Rock, Paper, or Scissors, and the computer randomly generates its choice. The program then compares both selections and determines whether the player wins, loses, or draws.

The game demonstrates the use of Python fundamentals such as conditional statements, lists, user input handling, and random number generation.

Features User selects an option using numbers: 0 → Rock 1 → Paper 2 → Scissors

Computer generates a random choice using the random module ASCII art is used to visually represent Rock, Paper, and Scissors The program evaluates the results and displays: Win Lose Draw Input validation ensures incorrect numbers are handled properly

Technologies Used Python Random module Conditional logic (if / elif / else) Lists User input

Learning Outcomes Through this project, I learned how to: Implement game logic using conditional statements Work with Python lists and indexing Use the random module to simulate computer decisions Handle user input and validate responses

Structure simple interactive command-line applications.
